How to connect our dashboard with your data

This quick tutorial introduces you to the process of connecting to your data. You can create as many data sources you want, but you only need one to get started.
1) Make a copy of this report 2) Connect a data source(s) to be added to the new report 3) If you start with sample data, then click on "Add data" icon 4) Select a default Google Ads connector for the type of data you want to visualize. 5) Authorize Data Studio to connect to your data. 6) Provide your account details and configure any additional options or parameters. 7) Confirm your choices to create a data source based on this connection. 8) Continue to Analytics page
